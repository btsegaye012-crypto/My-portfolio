# Personal Portfolio Website

This is a professional and responsive personal portfolio website built to showcase my skills as a Frontend Developer. The project focuses on clean code, modern design, and mobile-first responsiveness.

## 🚀 Features
* **Fully Responsive:** Works perfectly on mobile, tablet, and desktop screens.
* **Modern UI:** Designed with a focus on user experience and clean aesthetics.
* **Clean Code:** Built using semantic HTML5 and modern CSS best practices.

## 🛠️ Tech Stack
* **HTML5:** For the structure and semantic content.
* **CSS3:** For styling, Flexbox, and Grid layouts.

## 📁 Project Structure
* `index.html` - The main entry point of the website.
* `css/` - Contains the styling files (if applicable).
* `assets/` - Images and icons used in the project.
